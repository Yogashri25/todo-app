# API Documentation

Base URL: `http://localhost:5000`

All responses return JSON. All endpoints are prefixed with `/todos`.

---

## Todo Object Schema

```json
{
  "id": "uuid-string",
  "title": "Complete Assignment",
  "description": "Finish the React Todo App",
  "priority": "High",
  "status": "Pending",
  "dueDate": "2026-06-29",
  "createdAt": "2026-06-28T10:00:00.000Z",
  "updatedAt": "2026-06-28T10:00:00.000Z",
  "notes": "Push to GitHub before midnight."
}
```

**Field Reference:**

| Field       | Type   | Required | Description                            |
|-------------|--------|----------|----------------------------------------|
| id          | string | auto     | UUID generated by backend              |
| title       | string | ✅ Yes   | Short title of the task                |
| description | string | No       | Detailed description                   |
| priority    | string | No       | `"High"`, `"Medium"`, or `"Low"`       |
| status      | string | auto     | `"Pending"` or `"Completed"`           |
| dueDate     | string | No       | ISO date string (YYYY-MM-DD)           |
| createdAt   | string | auto     | ISO timestamp set on creation          |
| updatedAt   | string | auto     | ISO timestamp updated on every change  |
| notes       | string | No       | Additional notes                       |

## Endpoints

---

### GET /todos

Retrieve all todos.

**Request:**
```
GET /todos
```

**Response – 200 OK:**
```json
{
  "success": true,
  "count": 2,
  "data": [
    {
      "id": "abc-123",
      "title": "Learn React",
      "description": "Hooks and Context",
      "priority": "High",
      "status": "Pending",
      "dueDate": "2026-06-30",
      "createdAt": "2026-06-28T10:00:00.000Z",
      "updatedAt": "2026-06-28T10:00:00.000Z",
      "notes": ""
    }
  ]
}
```

---

### GET /todos/:id

Retrieve a single todo by ID.

**Request:**
```
GET /todos/abc-123
```

**Response – 200 OK:**
```json
{
  "success": true,
  "data": {
    "id": "abc-123",
    "title": "Learn React",
    "priority": "High",
    "status": "Pending",
    ...
  }
}
```

**Response – 404 Not Found:**
```json
{
  "success": false,
  "message": "Todo not found"
}
```

---

### POST /todos

Create a new todo.

**Request:**
```
POST /todos
Content-Type: application/json
```

**Request Body:**
```json
{
  "title": "Learn React",
  "description": "Study hooks and context API",
  "priority": "High",
  "dueDate": "2026-06-30",
  "notes": "Focus on useEffect and useState"
}
```

**Response – 201 Created:**
```json
{
  "success": true,
  "message": "Todo created successfully",
  "data": {
    "id": "new-uuid-here",
    "title": "Learn React",
    "description": "Study hooks and context API",
    "priority": "High",
    "status": "Pending",
    "dueDate": "2026-06-30",
    "createdAt": "2026-06-28T12:00:00.000Z",
    "updatedAt": "2026-06-28T12:00:00.000Z",
    "notes": "Focus on useEffect and useState"
  }
}
```

**Response – 400 Bad Request:**
```json
{
  "success": false,
  "message": "Title is required"
}
```

---

### PUT /todos/:id

Update a todo completely (all fields).

**Request:**
```
PUT /todos/abc-123
Content-Type: application/json
```

**Request Body:**
```json
{
  "title": "Learn React (Updated)",
  "description": "Deep dive into hooks",
  "priority": "Medium",
  "dueDate": "2026-07-01",
  "notes": "Also study Context API",
  "status": "Pending"
}
```

**Response – 200 OK:**
```json
{
  "success": true,
  "message": "Todo updated successfully",
  "data": { ...updated todo object... }
}
```

---

### PATCH /todos/:id/status

Toggle the completion status of a todo (Pending ↔ Completed).

**Request:**
```
PATCH /todos/abc-123/status
```

**Response – 200 OK:**
```json
{
  "success": true,
  "message": "Todo marked as Completed",
  "data": { ...updated todo object... }
}
```

---

### DELETE /todos/:id

Delete a todo permanently.

**Request:**
```
DELETE /todos/abc-123
```

**Response – 200 OK:**
```json
{
  "success": true,
  "message": "Todo deleted successfully",
  "data": { ...deleted todo object... }
}
```

**Response – 404 Not Found:**
```json
{
  "success": false,
  "message": "Todo not found"
}
```

---

## Error Responses

| Status | Meaning                        |
|--------|--------------------------------|
| 200    | Success                        |
| 201    | Created successfully           |
| 400    | Validation error (bad input)   |
| 404    | Todo not found                 |
| 500    | Internal server error          |

---

## CORS

The backend allows requests from `http://localhost:5173` (the Vite dev server). CORS is configured using the `cors` npm package in `server.js`.
